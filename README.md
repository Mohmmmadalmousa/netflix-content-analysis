# Netflix Data Analysis & Visualization

This is my **first data analysis project**, where I explored Netflix movies and TV shows using Python and pandas.  
The goal of this project was to **learn by doing**, and each step taught me something new — from cleaning data to creating visualizations and handling errors along the way.

I see this project as the **starting point of my journey** in data analysis. Every mistake I made (or will make) is a chance to improve and build better projects in the future.

---

## Project Overview

This project includes:

- Data cleaning and handling missing values  
- Exploratory Data Analysis (EDA)  
- Visualizations to uncover trends in Netflix content, including:  
  - Top 10 countries producing Netflix content (`top_countries.png`)  
  - Netflix releases per year (`releases_per_year.png`)  
  - Titles added per year (`titles_added_per_year.png`)  
  - Movies vs TV Shows breakdown (`movies_vs_tv.png`)  
  - Top genres (`top_genres.png`)  
- Saving charts as PNG images for easy sharing
